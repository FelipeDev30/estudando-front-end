Bem-vindo ao Chrome DevTools

O Chrome DevTools é um conjunto de ferramentas de desenvolvimento que permite depurar e otimizar seu código da web.

Com ele, você pode inspecionar elementos, visualizar o console, monitorar a rede e analisar o desempenho da sua aplicação.

Console

O console do Chrome DevTools é uma poderosa ferramenta para depuração e execução de comandos JavaScript.

Você pode usar o console para interagir com a página, inspecionar elementos e muito mais.

Rede

A aba de Rede do Chrome DevTools permite monitorar e analisar as requisições de rede feitas pela sua aplicação.

Você pode inspecionar cada requisição, visualizar os headers, o payload e a resposta do servidor.

Desempenho

A aba de Desempenho do Chrome DevTools permite analisar o desempenho da sua aplicação em tempo real.

Você pode visualizar métricas como tempo de carregamento, uso de CPU e memória, e identificar gargalos de desempenho.
