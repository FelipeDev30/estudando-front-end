Flexbox

O Flexbox é um modelo de layout unidimensional que facilita a distribuição de espaço entre os itens em um contêiner, mesmo quando seu tamanho é desconhecido ou dinâmico. Ele permite alinhar e distribuir os itens de forma eficiente, tanto na direção horizontal quanto na vertical.
